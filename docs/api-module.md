The complete Song of Heroic Lands public API — every exported class, interface, type, function, and constant, in one place.

Browse by architecture using the **categories** below (and in the sidebar):

- **Core** — Foundry-layer foundations: system registration, the data-model and logic base classes, the `FoundryHelpers` shim, calendar, and event queue.
- **Documents** — Foundry document classes by kind: Actor, Item, Combat, Combatant, Chat, Effect, Scene, Token.
- **Domain** — pure, Foundry-free game-mechanics objects: Modifier, Result, Body, Action, Movement, StrikeMode, SkillBase.
- **Utility** — shared helpers: Constants, Helpers, Collection, AI.
- **Applications** — standalone Foundry application windows.

New here? Start with the **Architecture Overview** and **Getting Started** guides linked from the home page, which explain the mental model and the `src/` layout this API mirrors.
