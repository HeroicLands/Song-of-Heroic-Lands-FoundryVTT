# Documentation

Guides and reference for **building against or extending** Song of Heroic
Lands. The **API Reference** (in the sidebar) documents every exported symbol;
the sections here explain how and why the system is built.

- **Concepts** — the mental model and architecture.
- **How-to** — task-oriented guides, from setup to house rules.
- **Reference** — contracts, catalogs, and subsystem deep-dives.
- **Contributing** — maintainer and project-meta.
