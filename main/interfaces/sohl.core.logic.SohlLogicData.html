Interface SohlLogicData<TParent, TLogic>

The base data interface for all Logic classes.

Every actor/item data interface (e.g., sohl.document.item.logic.SohlItemData, sohl.document.actor.logic.SohlActorData, sohl.document.item.logic.GearData) ultimately extends this. The corresponding SohlDataModel class implements it via Foundry's schema system.

Remarks

The base shape of every document's system data, reachable as document.system and (typed as the interface) document.logic.data.

interface SohlLogicData<
    TParent extends SohlDocument
    | SohlTokenDocument = SohlDocument,
    TLogic extends SohlLogic<any> = SohlLogic<any>,
> {
    actionDefs: sohl.entity.action.SohlAction.Data[];
    actorLogic: null | SohlActorLogic<any>;
    id: string;
    isOwner: boolean;
    kind: string;
    logic: TLogic;
    name: string;
    parent: null | TParent;
    shortcode: string;
    type: string;
    uuid: string;
    getFlag(scope: string, key: string): unknown;
    setFlag(scope: string, key: string, value: unknown): Promise<unknown>;
    update(data: object): Promise<unknown>;
}
