Interface TraitData<TLogic>

Remarks

The shape of system on a trait item — i.e. item.system (equivalently item.logic.data) when item.type === "trait". The backing DataModel implements this interface.

interface TraitData<TLogic extends TraitLogic<TraitData> = TraitLogic<any>> {
    actionDefs: SohlAction.Data[];
    actorLogic: null | SohlActorLogic<any>;
    choices: StrictObject<string>;

    id: string;

    isNumeric: boolean;

}
