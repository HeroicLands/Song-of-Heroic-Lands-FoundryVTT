Interface MovementProfile

A single per-medium movement profile persisted on a corpus. Bundles the being's speeds in one MovementMedium with the SafeExpressions (stored as source strings) that turn carried weight into encumbrance and shift it by strength.

interface MovementProfile {
    disabled: boolean;
    encumbrance: string;
    feetPerRound: number;
    leaguesPerWatch: number;
    medium: MovementMedium;
    strMod: string;
}
