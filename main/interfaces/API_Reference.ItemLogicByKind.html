Interface ItemLogicByKind

Item-kind → concrete Logic instance type, derived from ITEM_LOGIC_DEF.

Remarks

Lets callers resolve the precise logic type for a kind — e.g. ItemLogicByKind["skill"] is SkillLogic. Consumed by SohlActor.getItemLogic to return a concrete logic type from the item kind passed at the call site.

interface ItemLogicByKind {
    affiliation: AffiliationLogic;
    affliction: AfflictionLogic;
    armorgear: ArmorGearLogic;
    attribute: AttributeLogic;
    concoctiongear: ConcoctionGearLogic;
    containergear: ContainerGearLogic;
    corpus: CorpusLogic;
    miscgear: MiscGearLogic;
    mystery: MysteryLogic;
    mysticalability: MysticalAbilityLogic;
    projectilegear: ProjectileGearLogic;
    skill: SkillLogic;
    trait: TraitLogic;
    trauma: TraumaLogic;
    weapongear: WeaponGearLogic;
}
