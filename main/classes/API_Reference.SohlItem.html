Class SohlItem

Base class for all Item documents in the SoHL system — affiliations, afflictions, gear (armor, weapons, containers, misc, projectiles, concoctions), combat techniques, mysteries, mystical abilities, skills, traits, and traumas.

Like SohlActor, the typed game-rules surface lives on the item's logic object: prefer item.logic (equivalently item.system.logic) and the typed item.logic.data (SohlItemData) over reaching into item.system directly.
