Class SohlMersenneTwister

A seedable Mersenne Twister (MT19937) pseudo-random number generator.

Provides deterministic, reproducible sequences from a 32-bit integer seed — useful for tests and seeded world generation where Math.random would be non-reproducible. Instances may be created directly, or a process-wide singleton can be used via the static helpers (getInstance, setSeed, useMock, reset and the static int/random/ normal shortcuts).
