Class ContainerGearLogic<TData>

Logic for the Container Gear item type — storage for other items.

Container Gear represents backpacks, saddlebags, chests, belt pouches, cargo holds, and other receptacles that hold gear items. Containers track a maximum capacity limiting how much they can store.

Container Gear may be attached to Beings (a character's backpack) or Vehicles (a ship's cargo hold). Nested items inside a container inherit carry/equip state from the container.

Inherits weight, value, quality, and durability tracking from GearLogic.
