threatenedBy

  • get threatenedBy(): SohlCombatant[]

  • An array of combatants which are currently threatening this combatant. A combatant c threatens this one iff it is an enemy that is not defeated, not incapacitated (see THREAT_NEGATING_STATUSES), not hidden, and within weapon reach. This is useful for various combat mechanics, such as determining if a combatant is outnumbered or can be attacked in melee.

isEnemyOf

  • isEnemyOf(other: SohlCombatant): boolean

  • Pure relational predicate: two combatants are enemies iff they belong to different CombatantGroups. A combatant is never its own enemy, and an absent group on either side is treated defensively as enemy.

    Reads only already-loaded combatant fields — no Foundry API calls.

reaches

  • reaches(other: SohlCombatant): boolean

  • Whether this combatant's melee reach extends to other — i.e. the center-to-center grid distance between the two combatants' tokens is within this combatant's reach. Returns false when either token position is unavailable.

    Parameters

    • other: SohlCombatant

    Returns boolean

    Remarks

    Distance is measured center-to-center by design: a large creature's body size is folded into its lineage reachBase (a dragon has a large reach), so a big token's reach already accounts for the distance from its center to an adjacent target. Do not "fix" this to edge-to-edge.

    • Defined in src/document/combatant/SohlCombatant.ts:188
