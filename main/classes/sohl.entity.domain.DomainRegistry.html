Class DomainRegistry

Central registry for the world-scoped Domain settings.

All access to domain data — reads and writes — should go through this class. Logic-layer code must never call game.settings.get/set directly for the "sohl.domains" setting.

Staticregister

  • register(entries: DomainEntry | DomainEntry[], source?: string): Promise<void>

  • Register one or more entries. Merges into the existing store; if a shortcode already exists, the new entry overwrites it (last-write-wins).

    Entries with no source field are tagged with the explicit source argument if given, otherwise with "sohl" during the init hook or "world" after ready.
