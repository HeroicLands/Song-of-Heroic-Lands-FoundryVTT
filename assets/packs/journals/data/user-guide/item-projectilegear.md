# What Is Projectile Gear? {#projectilegear-overview}

Projectile Gear represents ammunition used with ranged weapons — arrows,
bolts, sling stones, javelins, or any other missile that is launched or
thrown in combat. Projectile Gear is consumed when used and is referenced
by Missile Weapon Strike Modes to determine which ammunition a ranged
attack uses.

# Where It Appears {#projectilegear-where}

Projectile Gear belongs to Beings. It appears on the Being sheet's **Gear**
tab and can be nested inside Container Gear items (such as a quiver). When
a character makes a ranged attack using a Missile Weapon Strike Mode, the
strike mode references the appropriate Projectile Gear to determine
ammunition availability and any damage modifiers the projectile provides.

# Key Properties {#projectilegear-properties}

- **Quantity** — how many projectiles the character has remaining.
- **Weight** — the weight per projectile (or per bundle), contributing
  to encumbrance.
- **Damage Modifier** — any bonus or penalty to damage that this
  projectile type provides.
- **Description** — notes about the projectile type and its properties.
