# What Is Miscellaneous Gear? {#miscgear-overview}

Miscellaneous Gear is the catch-all category for general equipment that
does not fall into the specialized categories of weapons, armor, containers,
projectiles, or concoctions. This includes tools, clothing, rope, torches,
food, trade goods, coins, and any other items a character might carry.
If an item is not a weapon, armor, container, ammo, or consumable potion,
it is Misc Gear.

# Where It Appears {#miscgear-where}

Misc Gear belongs to Beings. It appears on the Being sheet's **Gear** tab
and can optionally be nested inside a Container Gear item to reflect where
the character is carrying it. Misc Gear items are commonly added from
compendium packs or created manually to represent ad-hoc possessions.

# Key Properties {#miscgear-properties}

- **Weight** — how heavy the item is, contributing to encumbrance.
- **Quantity** — how many of the item the character possesses.
- **Value** — the monetary worth of the item.
- **Description** — notes about the item's purpose and properties.
