# Overview {#XvgvfmGmDVg3m3Hf}

This guide walks through creating a playable **Being** actor in SoHL.

See also: [Documentation Hub](README.md), [Quickstart](user-guide/quickstart.md), [Combat Basics](user-guide/combat-basics.md)

# Create the actor {#8qHUveVr9fydLyt2}

1. In the Actors directory, create a new actor.
2. Select actor type **Being**.
3. Set name, image, and any basic biography fields.

# Establish the core profile {#R3l5KuobhX1By87m}

Populate the essentials first:

- Traits
- Skills
- Movement profile
- Starting gear

A playable baseline is more useful than filling every field before first session.

# Add gear and nested capabilities {#ILz9m5IwXSps0g51}

Many SoHL items are compositional (for example, weapon entries with strike modes).

1. Add key gear (weapon, armor, utility equipment).
2. Verify expected nested items/modes are present.
3. Confirm action availability on the sheet.

# Add mystical or special features (optional) {#devAGWhv7poer90A}

If the character uses mystical systems:

1. Add relevant **Mystery** and/or **Mystical Ability** items.
2. Confirm associated skills or prerequisites are present.
3. Verify charges/levels where applicable.

# Validate sheet readiness {#W7Z866iVU9jpKmHo}

Before play, confirm:

- At least one combat-capable action is available.
- Skills and core traits are non-empty where needed.
- Encumbrance/movement values look reasonable.

# Common mistakes {#k28yWmvgQanQcD24}

- Wrong actor type selected (not **Being**).
- Missing nested strike mode on a weapon setup.
- Using incorrect base values for traits or skills.
