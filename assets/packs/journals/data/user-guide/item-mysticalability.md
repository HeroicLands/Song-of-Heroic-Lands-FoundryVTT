# What Is a Mystical Ability? {#mysticalability-overview}

A Mystical Ability is a specific supernatural power — a spell, prayer,
ritual, or other manifestation of mystical energy that a character can
invoke. Each Mystical Ability defines exactly what happens when it is
used: its effects, costs, range, duration, and any requirements for
successful invocation. These are the individual "spells" that make up
a practitioner's repertoire.

# Where It Appears {#mysticalability-where}

Mystical Abilities belong to Beings. They appear on the Being sheet's
**Mystical** tab. A Mystical Ability may reference a Mystery (the
tradition it belongs to) and a Domain (the sphere of authority it draws
upon). Mystical Abilities can also be nested inside Mystical Device items,
representing powers imbued into magical objects.

# Key Properties {#mysticalability-properties}

- **Name** — the name of the spell, prayer, or power.
- **Level** — the difficulty or power tier of the ability.
- **Associated Mystery** — which Mystery tradition this ability belongs to.
- **Associated Domain** — which Domain of mystical authority this ability
  draws upon.
- **Effect** — what happens when the ability is successfully invoked.
- **Cost** — any fatigue, piety, or other resource consumed by using
  the ability.
