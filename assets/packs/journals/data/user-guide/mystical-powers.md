# Overview {#mystical-overview}

Mystical powers in SoHL encompass all supernatural abilities — arcane
spellcasting, divine miracles, spiritual gifts, and other extraordinary
capabilities. The mystical system is organized into a hierarchy:
Philosophies contain Domains, which group Mysteries, which define
Mystical Abilities.

See also: [Philosophies](user-guide/item-philosophy.md), [Domains](user-guide/item-domain.md), [Mysteries](user-guide/item-mystery.md), [Mystical Abilities](user-guide/item-mysticalability.md), [Mystical Devices](user-guide/item-mysticaldevice.md)

# The Mystical Hierarchy {#mystical-hierarchy}

Understanding the relationship between mystical item types is key:

```
Philosophy (belief system)
  └── Domain (sphere of authority)
       └── Mystery (school/tradition)
            └── Mystical Ability (specific power/spell)
```

- A **Philosophy** is the overarching belief system (e.g., Arcane magic,
  Peonian faith)
- A **Domain** is a sphere within that philosophy (e.g., Fire magic,
  Healing miracles)
- A **Mystery** is a specific tradition or school within the domain
- A **Mystical Ability** is an individual spell, prayer, or power

A character's access to mystical abilities flows through this hierarchy:
they follow a Philosophy, which grants access to Domains, which contain
Mysteries with specific abilities.

# Casting Spells and Performing Miracles {#mystical-casting}

To use a mystical ability:

1. Open the character's sheet and go to the **Mystical** tab.
2. Find the ability you want to use.
3. Click the ability name to initiate a test.
4. The system resolves the test based on the ability's skill requirements.

# Mystical Devices {#mystical-devices}

A **Mystical Device** is a magical item that contains mystical abilities.
Enchanted weapons, holy relics, and magical rings are all mystical devices.
The device holds nested Mystical Abilities that the wielder can invoke.

When a Mystical Device is given to a character (by dropping it onto their
sheet), its abilities become available on the Mystical tab.

# The Mystical Tab {#mystical-tab}

The Mystical tab on a Being's sheet shows:

- **Philosophies** the character follows
- **Domains** the character has access to
- **Mysteries** the character has learned
- **Mystical Abilities** available to the character
