# What Is a Movement Profile? {#movementprofile-overview}

A Movement Profile defines how fast an actor can move and under what
conditions. It covers different movement modes — walking, running,
swimming, climbing, and so on — along with speed values and any terrain
modifiers that affect movement. Movement Profiles allow different creatures
and vehicles to have distinct mobility characteristics.

# Where It Appears {#movementprofile-where}

Movement Profiles belong to Beings, Cohorts, and Vehicles. A Being might
have profiles for walking and running, while a horse might also have a
galloping profile, and a ship would have sailing speeds. They are used
during play to determine how far a character can move on their turn and
how terrain affects travel.

# Key Properties {#movementprofile-properties}

- **Movement Mode** — the type of movement (e.g., walk, run, swim, climb,
  fly).
- **Speed** — the base movement rate for this mode.
- **Terrain Modifiers** — adjustments to speed based on terrain type or
  conditions.
