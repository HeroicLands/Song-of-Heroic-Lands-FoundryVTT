# What Is a Skill? {#skill-overview}

A Skill represents a learned or practiced ability that a Being possesses.
Skills cover everything from combat proficiency (Sword, Shield, Bow) to
everyday talents (Riding, Climbing, Awareness) to specialized knowledge
(Herblore, Law, Physician). Every meaningful action a character attempts
in the game is typically resolved through a skill test.

# Where It Appears {#skill-where}

Skills belong to Beings. They appear on the Being sheet's **Skills** tab,
organized by category (such as Combat, Physical, Communication, and so on).
Skills are typically added from compendium packs during character creation.

Each skill is based on one or more trait attributes via a **skill base
formula** — for example, a physical skill might derive its base from the
average of Strength and Dexterity. This means a character's innate
attributes directly influence their starting skill levels.

# Key Properties {#skill-properties}

- **Skill Base (SB)** — the starting value calculated from the character's
  trait attributes using the skill base formula. This is derived automatically.
- **Mastery Level (ML)** — the character's effective skill level, combining
  the skill base with any training, experience, and modifiers.
- **Skill Base Formula** — defines which attributes contribute to the skill
  base and how they are combined.
- **Category** — groups related skills together for organization on the
  character sheet.
