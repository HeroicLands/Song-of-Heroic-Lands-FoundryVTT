# What Are Assemblies? {#asm-what-are}

An Assembly is a container for complex items. While a normal World Item exists
on its own, an Assembly can hold an item along with everything nested inside it
-- strike modes, enchantments, contained gear, and more.

Think of an Assembly as a complete package: a weapon with all its attack modes,
an enchanted ring with its magical abilities, or a treasure chest full of loot.

# When to Use Assemblies {#asm-when-to-use}

Use Assemblies when you need an item that has nested components:

- **Weapons** with strike modes and enchantments
- **Enchanted gear** with mystical devices and abilities
- **Treasure chests** or bags containing other items
- **Any complex item** that needs to exist as a single, transferable unit

You do not need Assemblies for simple items. A plain sword or a set of clothes
can live as a regular World Item.

# The Assemblies Sidebar {#asm-sidebar}

Assemblies have their own tab in the sidebar, separate from Actors and Items.
Look for the box icon in the sidebar tab bar.

The Assemblies sidebar works like the Actors and Items tabs:
- **Create Assembly** button at the top to create a new Assembly
- **Create Folder** button to organize Assemblies into folders
- **Search bar** to filter by name

# Creating an Assembly {#asm-creating}

There are two ways to create an Assembly:

**From the Assemblies sidebar:**
1. Click the **Create Assembly** button.
2. A Create Item dialog appears -- choose the item type and name.
3. The item is created inside a new Assembly with the same name.
4. The Assembly appears in your sidebar, ready to edit.

**From the Items sidebar:**
1. Right-click any World Item.
2. Select **Create Assembly** from the context menu.
3. A new Assembly is created containing a copy of that item.

# Using Assemblies {#asm-using}

**Opening an Assembly:** Double-click an Assembly in the sidebar. The sheet
for its main item opens directly -- you interact with it as if it were a
normal item.

**Adding items to a character:** Drag an Assembly from the sidebar onto a
character sheet. All items inside the Assembly (including nested items) are
copied onto the character with their relationships intact. The result is the
same as if you had added each item by hand.

**Placing on a scene:** Drag an Assembly from the sidebar onto the canvas
to create a token. This is useful for treasure chests, interactive objects,
or items that exist in the game world as physical things players can find.

# Examples {#asm-examples}

**Enchanted Broadsword:**
An Assembly containing a Broadsword (the main item), with nested Slash and
Thrust strike modes, plus an enchantment device with magical abilities.
Drag it onto a character to give them the complete weapon.

**Treasure Chest:**
An Assembly containing a Container (the main item), with coins, potions,
and equipment nested inside. Place it on a scene as a token that players
can interact with and loot from.
