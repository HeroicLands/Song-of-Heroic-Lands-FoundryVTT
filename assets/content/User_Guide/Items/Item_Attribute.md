# What Is an Attribute?

An attribute represents a fundamental intrinsic characteristic of a Being, something about what they are. Traits cover a wide range: physical like Strength or Dexterity, psyche like Reasoning or Eloquence, and transcendant qualities such as Aura. These serve as the foundation for skill calculations throughout the system.

# Where It Appears

Attributes appear on the Being sheet's **Profile** tab.

Attributes are displayed prominently since they form the basis of skill base formulas — every skill derives its starting value from one or more attributes.

Attributes are typically added from compendium packs when creating a character,
not created from scratch.

# Additional Properties
Along with the [[Item_Base|Standard Item Properties]], the following properties also appear in the **Properties** tab:
- **SubType:** Traid category ("Physical", "Personality", "Transcendant")
- **init Dice Formula:** If numeric, The dice formula that is used when a new being is created.  If empty, the existing value will be kept.
- **Value Descriptions** — If numeric, this is a table of descriptions for various trait scores.
- **Score:** Value of the trait.
