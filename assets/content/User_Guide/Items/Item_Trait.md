# What Is a Trait?

A Trait represents characteristic of a Being (with the exception of attributes, which are tracked in [[Attributes]]) — something about who or what they are. Traits cover physical like missing limbs or scars and psyche like phobias or psychoses.

# Where It Appears

Traits appear on the Being sheet's **Profile** tab. 

# Additional Properties
Along with the [[Item_Base|Standard Item Properties]], the following properties also appear in the **Properties** tab:
- **SubType:** Traid category ("Physical", "Personality", "Transcendant")
- **Intensity** — The impact or severity of the trait:
  - **Trait:** Simply a characteristic of the character, but no compulsion.
  - **Impulse:** A minor compulsion, observable after some time interacting with the Being, but which does not generally affect the daily process of living.
  - **Disorder:** A major compulsion, observable after a brief time interacting with the Being, that generally affects the daily process of living.
- **Is Numeric:** If the value of this trait is numeric or not.
- **Score:** If numeric, the score will be displayed
  - **Value:** The numeric value of the score
  - **Max:** The maximum value of the score; 0 indicates no maximum
- **Text Value:** Textual Value of the trait, only shown if non-numeric.
- **init Dice Formula:** If numeric, The dice formula that is used when a new being is created.  If empty, the existing value will be kept.
- **Choices:** If not numeric, the set of values this field can contain. If none are specified, the field is free-form and any value may be supplied.
- **Value Descriptions** — If numeric, this is a table of descriptions for various trait values.
