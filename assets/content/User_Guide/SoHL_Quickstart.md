This quickstart gets a user from install to first roll with Song of Heroic Lands (SoHL).

# Install and Enable

1. Install the SoHL system in Foundry VTT.
2. Create a world and select **Song of Heroic Lands** as the game system.
3. Launch the world.

# Create a First Character

Generally speaking, you should avoid using the Create Actor dialog for Beings. It will require
setting up a number of items by hand, and is very difficult. The suggested mechanism is to
duplicate an existing Being, and modify it as needed.

1. Go into the Beings compendium, and import the "Basic Folk" Being.
2. Double-click on the Being to open the character sheet.
3. In the header, click on the Basic Folk name and rename to something else.
4. Click on the Profile tab.
5. For each attribute, click on the edit button to the upper right of the attribute value, and change the value as appropriate.
6. Click on the Skills tab.
7. For each initial skill, right click on the skill and choose "Edit" from the context menu. Setup the initial Mastery Level value for the skill.
8. Click on the Gear tab.
9. Open the Weapons compendium, and drag over a broadsword and a roundshield, dropping them onto the character sheet. The items should show up on the Gear tab.
10. Click on the Combat tab. Note that there are no weapons showing. Now click on the Gear tab. Click on the shield ("equip") icon for both the Broadsword and the Roundshield. Go back to the Combat tab. The Roundshield and the Broadsword should now show up under weapons.

# Perform a Skill Roll

1. In an open character sheet, click on the Skills tab.
2. Click on the name of a skill. A Success Test Dialog should appear. Simply press OK to perform the roll. The result should show in the chat window.

See [[Combat_Basics|Combat Basics]] for round flow and common checks.
